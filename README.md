# ALIMAZ Pension Website

Premium boutique guest house website for ALIMAZ Pension in Wolayta, Ethiopia.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Images

Add the client's real photography to `public/images` using the filenames listed in `public/images/README.md`.

The current implementation avoids third-party stock images so the final site can follow the client image policy.
